NetBox (Source of Truth)

Loading… Open NetBox ↗

NetBox acts as the network source of truth. Each device list is mirrored as a NetBox region (with a same-named site beneath it). Every device is scanned via SSH to pull model, serial number, platform, and software version, and its IPv4 interface addresses are pushed into NetBox IPAM as prefixes and IP addresses assigned to DCIM interfaces — with the management IP set as the device's primary IPv4.

NetBox is not configured

To use this tab, enter your NetBox server URL and API token in Settings. The API token must be a NetBox API key (see Admin → API Tokens in NetBox). This app authenticates using the Authorization: Bearer <token> header NetBox's own REST API docs recommend, and falls back to Token <token> for older NetBox versions.

Sync

Syncing connects to every online device in the selected list(s), pulls show version, show inventory, and show ip interface, then pushes the results into NetBox DCIM + IPAM. Each list becomes its own region. Unreachable devices are reported but not created in NetBox.
